# EU Compliance

## EUPL-1.2

Skatteprogressivitet is licensed exclusively under the European Union Public Licence v. 1.2
(EUPL-1.2). All contributions must be compatible with the EUPL-1.2 compatible-licence list
in the Appendix. No dual licensing is offered.

## EC OSS Strategy 2020-2023 (C(2020)7149 final)

This repository follows the EC Open Source Software Strategy 2020-2023 principles:
think open, transform, share, contribute, and secure.

## OSOR Good Practice

The project is registered with the Open Source Observatory (OSOR) and follows its
good-practice catalogue for open-source research software in public administrations.

## Interoperable Europe Act (EU) 2024/903

The repository is designed for interoperability across EU member-state research
infrastructures. A submission to the Interoperable Europe Portal is planned within
30 days of the first stable release.

## REUSE 3.0 (DEP5 Only)

Copyright and licence declarations are managed exclusively via `.reuse/dep5`.
No per-file SPDX comment headers are used. REUSE compliance is verified by `reuse lint`
in CI on every push and pull request.
